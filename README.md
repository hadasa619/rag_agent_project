# 🤖 RAG Agentic Coding Assistant

יישום RAG מתקדם המאחד תיעוד טכני ממספר כלי Agentic Coding לשכבת ידע אחת אחודה.

## 📌 עיקרי הפרויקט
* **איחוד ידע:** סריקה ואנדוקס של קבצי Markdown מתיקיות פרויקט שונות.
* **ארכיטקטורת Workflow:** ניהול תהליך מבוסס אירועים (Events) ושלבים (Steps) באמצעות LlamaIndex.
* **נתב חכם (Router):** ניתוח שאלת המשתמש וניתוב בין חיפוש סמנטי (Semantic) לשליפה מובנית (Structured).
* **חילוץ נתונים (Extraction):** יצירת שכבת מידע מובנית (JSON) עבור חוקים, החלטות ואזהרות לדיוק מקסימלי.

## 🚀 איך מריצים?
1. הגדרת מפתח API של Cohere בקובץ `.env`.
2. הרצת `extractor.py` ליצירת שכבת הנתונים המובנית.
3. הרצת `app.py` לפתיחת ממשק ה-Gradio.
4. לצפייה בארכיטקטורת התהליך, ניתן לפתוח את הקובץ `workflow_graph.html`.

## 🙋 דוגמאות לשאלות שהסוכן יודע לענות

הסוכן מזהה את סוג השאלה ובוחר במסלול האופטימלי:

### 📊 שליפה מובנית (מסלול המבטיח כיסוי מלא של רשימות)
* "תן לי רשימה של כל הכללים בפרויקט"
* "אילו החלטות טכניות התקבלו עד כה?"
* "הצג את כל האזהרות הקיימות במערכת"

* "מה הקונספט של עיצוב ה-RTL בפרויקט?"
* "איך מוגדר ה-Date Picker בממשק עברי?"
