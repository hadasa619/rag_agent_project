# סיכום החלטות ממשק ושפות
**כלי:** Claude Code
**נושא:** לוקליזציה (Localization)

## תרגומים מאושרים
- הוחלט סופית: המערכת תתמוך ב-3 שפות בגרסת ה-MVP:
    1. עברית (Hebrew) - תמיכה מלאה ב-RTL.
    2. אנגלית (English) - תמיכה ב-LTR.
    3. ערבית (Arabic) - תמיכה מלאה ב-RTL.

## הנחיות RTL נוספות
- סרגל הניווט (Navbar) יעבור לצד ימין בממשק עברי וערבי.
- לוגו החברה יופיע בצד ימין למעלה בשפות RTL ובצד שמאל למעלה בשפות LTR.
