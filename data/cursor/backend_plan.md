# תוכנית פיתוח Backend
**כלי:** Cursor Agent
**סטטוס:** טיוטה לביצוע
**גרסה:** 1.2

## הגדרות API
- כל ה-Endpoints של המערכת יחזירו תשובות במבנה JSON אחיד.
- יש להשתמש ב-Pydantic עבור ולידציה של נתונים נכנסים.

## אבטחה (Security)
- אימות משתמשים יתבצע באמצעות JWT (JSON Web Tokens).
- **חשוב:** תוקף ה-Token יהיה ל-24 שעות בלבד.

## הערות פיתוח
- החלטנו להוסיף תמיכה ב-Redis עבור Caching של שאילתות כבדות ב-PostgreSQL כדי לשפר ביצועים.
